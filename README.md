# Conversor de Temperatura (°C ➜ °F)

Um aplicativo simples desenvolvido em **React Native** utilizando **Expo**, com o objetivo de converter graus Celsius (°C) para Fahrenheit (°F).  
Projeto criado para praticar conceitos fundamentais como manipulação de estado, validações, componentes básicos e exibição de alertas nativos.

---

## Funcionalidades

✔ Conversão rápida de °C para °F  
✔ Tratamento de erros (campo vazio ou valor inválido)  
✔ Alerta nativo exibindo o resultado  
✔ Reset automático do campo após a conversão  
✔ Interface simples e clara  
✔ Lógico e direto ao ponto — ideal para estudos

## O que foi praticado neste projeto

- Uso de `useState` para gerenciar dados
- Validação de entrada usando `isNaN` e `.trim()`
- Converter strings com vírgula para números válidos
- Lidar com `Alert.alert()` do React Native
- Criação de layout com `StyleSheet`
- Estrutura básica de um pequeno app em React Native
- Organização de componentes e estilos

---

## Tecnologias Utilizadas

- **React Native**
- **Expo**
- **JavaScript ES6**
- **Componentes nativos** (`View`, `Text`, `TextInput`, `TouchableOpacity`, `Alert`)
